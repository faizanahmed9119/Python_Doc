Introduction

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting.
Hello World

To get started with Python, open an IDE and write your first "Hello world" Python code:

print("Hello World")
Variables

You use variables as symbolic names for values in your application.

You can declare variables in the following way where "x" is a variable

x = 24
Data Types

Data types are classifications of data that tell a compiler or interpreter how a programmer intends to use the data.

Following are the data types used in Python:

  • String: Used for a combination of characters, such as letters, numbers, and symbols
  • Integer: Used for whole numbers, including negative and positive numbers, and zero
  • Char: Used to represent a singular ASCII character
  • Floating Point: Used to store numbers that may have a fractional component, like monetary values
  • Boolean: Used to represent the values true and false, which are often used to clarify conditional statements
If statements

If statements are used to execute a statement if a logical condition is true in the following way:

a = 20
b = 30

if b > a:

(logical statement)

In the above example if the value of b is greater than a the logical statement will be executed.

While loops

A while statement executes its statements as long as a specified condition evaluates to true. A while statement looks as follows:

while (condition):

If the condition becomes false, statement within the loop stops executing and control passes to the statement following the loop.

The condition test occurs before statement in the loop is executed. If the condition returns true, statement is executed and the condition is tested again. If the condition returns false, execution stops and control is passed to the statement following while.
